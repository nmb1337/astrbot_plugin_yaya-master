# YaYa 关键词回复 🐟

群聊关键词自动回复插件，支持多图片回复、群聊白名单，以及 AI 大模型回复联动控制。

## 👤 作者

**贾梦** — 一个爱敲代码的咸鱼 🐟

## ✨ 功能

- 🎯 **关键词自动回复**：在插件配置面板添加关键词规则，群内有人发送包含关键词的消息时自动回复
- 🖼️ **多图片支持**：每条规则可配置多张图片 URL（支持 http/https 链接和本地路径）
- 📝 **自定义回复文本**：可自由编辑每条规则的回复文本
- 🔒 **群聊白名单**：配置群号白名单，仅白名单内的群聊触发此插件
- 🤖 **AI 联动控制**：可在配置面板选择触发关键词时是否同时使用 AI 大模型回复

## 📦 安装

在 AstrBot WebUI 插件市场中搜索「YaYa 关键词回复」安装，或手动将本插件放入 `data/plugins/` 目录。

## ⚙️ 配置

插件安装后，在 AstrBot WebUI 的插件管理 → YaYa 关键词回复 → 配置面板中进行设置：

| 配置项 | 说明 |
|--------|------|
| **关键词规则列表** | 添加多条关键词规则，每条包含「触发关键词」「回复文本」「回复图片URL列表」 |
| **群聊白名单** | 每行一个群号，留空则所有群聊生效 |
| **是否使用 AI 回复** | 开启后，触发关键词时 AI 大模型也会同时回复 |

## 🚀 使用示例

1. 在配置面板添加规则：关键词 = `菜单`，回复文本 = `这是本群的菜单：`，图片 = `https://example.com/menu.jpg`
2. 在群聊白名单中填入你的群号
3. 群内有人发送包含「菜单」的消息时，机器人自动回复文本和图片
