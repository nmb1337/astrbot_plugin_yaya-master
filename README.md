# YaYa 多级关键词菜单 🐟

多级关键词菜单回复插件，支持在后台构建树形菜单结构。群内用户输入根关键词进入菜单，通过**序号**或**关键词**逐级导航，最终获取文本 + 多图片回复。

## 👤 作者

**贾梦** — 一个爱敲代码的咸鱼 🐟

## ✨ 功能

- 🌳 **树形菜单**：在 Plugin Page 中构建多级关键词树（如：菜单 → 经理/副经理 → 财政/卫生/安保/售后）
- 🔢 **序号导航**：用户输入数字序号快速选择
- 🔤 **关键词导航**：用户输入子关键词直接跳转
- ↩️ **返回/退出**：发送「返回」回到上一级，「退出」关闭菜单（5 分钟超时自动退出）
- 🖼️ **多图片支持**：叶子节点可配置多张图片 URL
- 🔒 **群聊白名单**：在配置面板设置群号白名单
- 🤖 **AI 联动控制**：可选择触发回复时是否同时使用 AI 大模型

## 📦 安装

在 AstrBot WebUI 插件市场中搜索「YaYa 关键词回复」安装。

## ⚙️ 配置

### 基础配置（配置面板）

| 配置项 | 说明 |
|--------|------|
| **群聊白名单** | 每行一个群号，留空则所有群聊生效 |
| **是否使用 AI 回复** | 开启后触发回复时 AI 也会同时回复 |

### 菜单树管理（Plugin Page → keyword-console）

- 🌳 **树形视图**：折叠/展开查看完整菜单树
- ➕ **添加子节点**：点击任意节点的「+子」按钮添加下级
- ✏️ **编辑节点**：修改关键词、回复文本、图片
- 🗑️ **删除节点**：二次确认删除（含所有子节点）

## 🚀 使用示例

### 后台配置
```
📁 菜单 (请选择：)
  📁 经理 (经理下属部门：)
    📄 财政 (财政部电话：xxx，地址：xxx)
    📄 卫生 (卫生部电话：xxx)
  📁 副经理 (副经理下属部门：)
    📄 安保 (安保部电话：xxx)
    📄 项目售后 (售后电话：xxx，图片：xxx.jpg)
```

### 群内交互
```
用户: 菜单
Bot:  请选择：
      1. 经理
      2. 副经理
      ━━━━━━━━━━
      发送 返回/0 回到上一级 | 发送 退出 关闭菜单

用户: 1  (或 经理)
Bot:  经理下属部门：
      1. 财政
      2. 卫生
      ━━━━━━━━━━
      发送 返回/0 回到上一级 | 发送 退出 关闭菜单

用户: 财政
Bot:  财政部电话：xxx，地址：xxx
      [图片]
```
